## Задача 3

- Тип задачи: Web-scrapping

Краткое описание задачи
Задача заключается в парсинге Pinterest. Целью является автоматическое скачивание фотографий в data и составление dataset.csv, в котором будет путь до изображения, ссылка на изображение, стиль человека, tag, описание, автор поста, количество лайков, кличество комментариев, метка о скачивании и информация об ошибке. Также добавлена пагинация.

В первый заход нужно будет лично авторизоваться на Pinterest. В последующие информация о входе будет сохранена.

## Development


- Зависимости
```bash
pip install -r requirements.txt
```
